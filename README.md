TECH STACK USED
------------------------------------------------------------
- React JS

------------------------------------------------------------
PROJECT DESCRIPTIONS
------------------------------------------------------------

1. Shopping Cart
   - A complete shopping cart application demonstrating state management and component communication.
   - Features:
       * Add and remove products
       * Display total items
       * Display total price
       * Reusable components (ProductList, CartItem, CartSummary)

2. Greeting App
   - A form-based React application with validation.
   - Features:
       * Fields: name, email, password, confirm password
       * Real-time input validation
       * Error messages
       * Success message on submit
       * Submit button is disabled until all inputs are valid

3. Product Catalog
   - A simple product listing interface.
   - Features:
       * Displays list of products
       * Reusable ProductCatalog component
       * Uses .map() for dynamic rendering

4. Fruit App
   - A simple fruit information display application.
   - Features:
       * Shows a list of fruits
       * Allows selection of fruit
       * Displays fruit details dynamically

5. Weather Dashboard
   - A structured weather information application UI.
   - Features:
       * Input field for city name
       * Weather display structure
       * Reusable components
       * API integration

------------------------------------------------------------
HOW TO RUN THE PROJECT
------------------------------------------------------------

1. Clone the repository:
   git clone https://github.com/Tanushyadav/All-Cepialabs-Assignments-.git

2. Navigate into the project:
   cd All-Cepialabs-Assignments-

3. Install dependencies:
   npm install

4. Start the development server:
   npm start
